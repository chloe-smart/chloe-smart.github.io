About Me

PHP OOP API - Project Management API

Using OO PHP we created a project management API using a MySQL database, following the SOLID principles. Our API produced JSON, which was sent to the pre-built front-end. I worked on creating the application architecture diagram, building an entity class, throwing custom exceptions, a database connection class & a class that formatted the response, as well as writing unit tests.

JavaScript game - Typing Speed Game

We used JavaScript to make this typing speed game, fetching the words to type from a pre-existing RESTful API. We used event listeners to start the timer when the user starts typing, using setInterval / setTimeout to create the countdown / timer. I worked on the styling of the game, changing the colour of the current word depending if it was correct using DOM manipulation, calculating and displaying the results of the game with a modal pop up.
